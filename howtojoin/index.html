Join Bioschemas!

Bioschemas is an open community and welcomes any organization or individual to join this community effort. If you want to be involved in Bioschemas, you can have a look at the resources below.

New Types

If you are interested in having a new class/type in Bioschemas, please visit the New Types page, in which the whole process for the definition of a new Bioschemas type is fully described.

Google Groups

You can request to access one of these groups in order to receive the latest updates about a specific Bioschemas project.

  • Event
  • Training Material
  • Organization
  • Person
  • Standard
  • Commons
  • Community

GitHub pages

You can take part in the discussion about a specific Bioschemas project by joining one of these GitHub pages.

  • Event
  • Training Material
  • Organization
  • Person

Email

If you want to get in touch quickly with one of Bioschemas groups, you can send an email to one of the following addresses.

  • event@bioschemas.org
  • training-material@bioschemas.org
  • organization@bioschemas.org
  • person@bioschemas.org
  • standard@bioschemas.org
  • commons@bioschemas.org
  • community@bioschemas.org
  • all@bioschemas.org (for questions regarding the whole Bioschemas project)

If you missed a meeting, please check the Meetings web page.

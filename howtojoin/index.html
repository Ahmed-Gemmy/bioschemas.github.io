Join Bioschemas!

Bioschemas is an open community and welcomes any organization or individual to join this community effort. If you want to be involved in Bioschemas, please

  • Join the W3C Bioschemas Community Group (detailed instructions below)
  • Subscribe to the mailing list
  • Get involved in discussions on GitHub

W3C Community Group

Joining a W3C Community Group requires a W3C participant account, there is no need to be part of a membership organisation. Complete all sections of the form and then make the appropriate employment declaration on the second page. Once you have received the confirmation email you should be able to log in to your account and join the Bioschemas Community Group. More detailed step-by-step instructions are available here.

Mailing List

The Bioschemas mailing list is hosted by the W3C Bioschemas Community Group. There is no requirement for a W3C account to subscribe to the mailing list.

All messages to the mailing list are available through a public archive.

GitHub

We use GitHub to collaboratively develop new specifications, and tools to support the Bioschemas community. There are several project specific repositories as well as:

  • Bioschemas repository: for tracking tasks, ideas, issues, and opportunities
  • bioschemas.github.io for managing this website

Data Group

The BD2K NIH bioCADDIE project is working towards a DataMed prototype, to support data search as PubMed facilitates literature search.

The bioCADDIE Working Group 3 has worked on defining the dataset description model DatA Tag Suite (DATS) model specification, which is a model for describing datasets and enabling their discovery.

Now, the group is working on including the mapping of the DATS model to schema.org to extend the DATS JSON schemas into JSON-LD, as part of BioSchemas.
